# 行为准则

[English Version](docs/CODE_OF_CONDUCT_en.md)

## 我们的承诺

我们致力于为每一位参与者提供一个开放、友好、包容的社区环境，不论其年龄、体型、残疾状况、民族、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、宗教信仰或性取向如何。

## 我们的标准

有助于营造积极社区环境的行为包括：

- 使用友善、包容的语言
- 尊重不同的观点和经历
- 接受建设性的批评意见
- 关注对社区整体有益的事情
- 对其他社区成员保持善意和同理心

不可接受的行为包括：

- 使用带有性意味的语言或图像，以及任何形式的性骚扰
- 发表挑衅性言论、侮辱性或贬低性评论，以及人身或政治攻击
- 公开或私下骚扰他人
- 未经明确许可，发布他人的私人信息（如实际地址或电子邮件地址）
- 其他在专业场合中被合理认为不当的行为

## 执行责任

项目维护者有责任解释和执行本行为准则所定义的可接受行为标准，并对任何被认为不当、威胁性、冒犯性或有害的行为采取适当且公正的纠正措施。

项目维护者有权利和义务删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、issue 及其他贡献，并在适当时就处理决定的原因作出说明。

## 适用范围

本行为准则适用于所有项目空间，也适用于个人在公开场合代表本项目时的行为。代表项目的行为包括：使用项目的官方电子邮件地址、通过官方社交媒体账号发帖，或在线上/线下活动中担任指定代表等。

## 执行

如遇到辱骂、骚扰或其他不可接受的行为，请通过项目的 GitHub Issues 或直接联系维护者进行举报。所有投诉都将被认真审查和调查，并作出与实际情况相适应的回应。项目团队有义务对举报者保密。
